# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.x     | :white_check_mark: |

## Reporting a Vulnerability

Report via [GitHub Security Advisories](https://github.com/kopai/kopai-mono/security/advisories).

Do NOT open public issues for security vulnerabilities.

We do not operate a bounty program.
