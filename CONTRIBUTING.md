# Contributing

## Setup

```bash
pnpm install
pnpm build
```

## Development

```bash
pnpm dev  # watch mode
```

## Changesets

When adding features or fixing bugs:

```bash
pnpm changeset
# Select packages, version bump, write summary (past tense)
# Commit the .changeset/*.md file with your PR
```

## Testing

```bash
pnpm test
pnpm type-check
pnpm lint
```
